# Nike Store

A modern e-commerce frontend inspired by Nike, built with React, TypeScript, and Vite. The application provides a fast, responsive, and user-friendly shopping experience.

## Features

- Responsive modern design
- Product listing
- Product details page
- Shopping cart
- Favorites page
- Search functionality
- Login page
- Clean and reusable React components

## Technologies Used

- React
- TypeScript
- Vite
- CSS
- React Hooks

## Installation

`bash
npm install
npm run dev
